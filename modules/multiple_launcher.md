# Multiple Launcher
**Multiple Launcher** とは、複数のファイルやパスを一気に開く仕組みである。

たとえば「朝一番に開きたいアプリやファイル」が計 5 個あるとき、これをすべて開くランチャーをつくっておいて、実際開くときはランチャーを一回開くだけでいい。一から 5 個分を開くよりも楽である。この考え方で、「複数開きたい場合」を特定し、ランチャー化していくのだ。これにより、タスクその他行動を始める際の、最初のハードルを下げることができる。手作業で一つずつ起動するのと、ほぼ自動で一気に開くのとでは、ハードルの高さが全然違う。

ランチャーの実装方法は様々だ。バッチファイルやシェルスクリプトでつくってもいいし、Python のような LL で書いてもいい。AutoHotkey のようなスクリプトエンジンもありえるだろう。Windows と Mac など OS レベルでも最適なやり方は異なるはずだ。可能であれば、独自の拡張子を設定して、関連付けも設定した上で、そのファイルを直に開けるようにしたい。
